This project is a responsive login page UI built using HTML5, CSS3, and Bootstrap 5.
The design focuses on clean typography, modern colors, and user accessibility. It includes essential authentication elements like Email, Password, Remember Me, Forgot Password link, and Google Sign-in button.
A right-side image section and centered layout enhance the user experience and create a professional look. The page is fully mobile-responsive and adapts perfectly to all screen sizes.
